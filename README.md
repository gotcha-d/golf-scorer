# ゴルフスコア計算アプリ
## 目的
### オブジェクト指向のプログラミングスタイルを身体知として身につける
- 『ThoughtWorksアンソロジー』 ThoughtWorks Inc.出てくる9つのルール。
- その９つのルールを体に覚えさせるエクササイズをオブジェクト指向エクササイズと言う。
   - やや極端奈コーディングスタイルでOOPのコーディングスタイルを身につける。
- ９つのルールとは以下の通りである。
  - １つのメソッドにつきインデントは１段階までにすること
  - else 句を使用しないこと
  - すべてのプリミティブ型と文字列型をラップすること
  - １行につきドットは１つまでにすること
  - 名前を省略しないこと
  - すべてのエンティティを小さくすること
  - １つのクラスにつきインスタンス変数は２つまでにすること
  - ファーストクラスコレクションを使用すること
  - Getter, Setter, プロパティを使用しないこと
  - １つのメソッドにつきインデントは１段階までにすること

## ドメイン分析の結果の中間成果物
### イベント候補
#### 開始フェーズ
- コースが選ばれた
- プレイ種別が選ばれた
- ティーが選ばれた
これらの結果、ラウンドが始まった

#### プレイ中フェーズ
- ホールスコアが記録されたえ
- ホールスコアが訂正された

#### 終端フェーズ
- ラウンドが終了した
- ラウンドが破棄された

### ユビキタス言語
- ラウンド
  - 1回分のプレー単位
- フルプレー 18ホール（前半9, 後半9）
- ハーフプレー
- 前半9・後半9(OUT/IN)
- 中断
  - プレー中で止まること
